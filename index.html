YUZUMINT OS

Yuzu flavoured Linux Mint

YUZUMINT OS is a customized version of Linux Mint with a bunch of software preinstalled with the operating system. It's purpose was to allow me to quickly install everything I need onto my computer due to frequent Disk Failure using way cheap 2nd hand Hard Disks.

Why didn't I just use Windows? I was poor and my pc only costed $35 in total and I wasnt going to pay $100 to use Windows. If I had $100 would have bought a better 2nd hand PC.

Vtuber Noises

A Massive Success

Vtuber Noises started with the idea of replicating the Tuturu app but instead to have Uruha Rushia say "Not Pettan". It started as a C++ console application to a c# Windows form application to a HTML website to a Xamarin form Android application.

All of these failed terribly due to slow loading times, terrible audio, and terrible design. It was also not providing any reach as no one was downloading or using my apps. So I finally gave up and decided to learn Java and finally succeeded with my 1st published app Kiara Noises.

VN Automation

The future of Vtuber Noises

Vtuber Noises Automation is a C++ program that was developed to help me code at a faster rate to develop more Vtuber Noises based apps. Previously, I was doing everthing on my own manually. Writing every single line of code and editing the audio and images. This meant long hours of work and typos that often lead to errors in the code.

Ever since it was developed, it has been able to help me all of the code for Vtuber Noises based app. This allows me to have more time to edit audio and images and fine tune the UI and spend more time on my other projects.
